1. Abstract

iNews is a HTML website dedicated to help users to add and elimiate chosen articles/news based on RSS Feed.

2. Introduction

iNews compiles news feeds from a variety of online sources such as reddit for the user to customize and share with others.

3. Structure

iNews has a simple interface, once you are on the site, you are presented with our top chosen articles and news.

If the user is new he can create a new account. Once loged in he can add and remove specific subbredits for him to see.

The user has the accessibility to share any topic on his social media. The website also offers a few themes for the user choice.

3.1 The HomePage

Offers you the possibility to read the latest news presented in a nice way. If you have an account and you are connected, you can get only the news that you want by setting them in your page. You can do that by accesing your account if you press 'Edit your account' from header. Also, on the left side, you have a list of categories. Pressing one of them it will display you only the news from that category. Furthermore, anytime you want to access the Front Page, you can press our logo button from header, returning you to all news gathered togheter. If you access any article, it will send you to the Article Page or to an external url with more details.

3.2 The About Page

About.html shows a list of every html page and teir functinalities.

3.3 Sign Up Page

Offers you the possibility to create an account if you don't already have one. You can access this page by pressing the Sign Up button from the Front Page. When creating an account, you must complete several fields: first name and last name (these fields must contain only upper/lower letters, whitespaces or additional dash symbols), email (valid email format), username (not allowed to start with number or symbols; it must contain lower/uppers letters, numbers and aditional symbols : ".", "-", "_") and password( it must contain minimum 6 characters: at least one lower and one upper letter and at least one number). All these fields are required. If you want to go back to Front Page, press the logo button.

3.4 Log In Page

Submits your personal information (username and password) searching for an existing account in our database.

3.5 User Page

Offers you the possibility to customize your settings: update your personal information, choose the categories that you prefer and change your theme or the time you want us to refresh your news. If you want to go back to Front Page, press the logo button.

4. What is used.

The website is written in html, css and javascript. It is hosted with mongoDB. etc etc.

4.1 HTML

Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web.

4.2 Mongo Db

MongoDB is a free and open-source cross-platform document-oriented database program. Classified as a NoSQL database program, MongoDB uses JSON-like documents with schemas. MongoDB is developed by MongoDB Inc.

4.3 Json

JSON (JavaScript Object Notation) is a lightweight data-interchange format. It is easy for humans to read and write. It is easy for machines to parse and generate. It is based on a subset of the JavaScript Programming Language, Standard ECMA-262 3rd Edition - December 1999. JSON is a text format that is completely language independent but uses conventions that are familiar to programmers of the C-family of languages, including C, C++, C#, Java, JavaScript, Perl, Python, and many others. These properties make JSON an ideal data-interchange language.

4.4 Socket.io library

Socket.IO is a JavaScript library for realtime web applications. It enables realtime, bi-directional communication between web clients and servers. It has two parts: a client-side library that runs in the browser, and a server-side library for Node.js. Both components have a nearly identical API. Like Node.js, it is event-driven.

4.5 RSS Feed

RSS (Rich Site Summary; originally RDF Site Summary; often called Really Simple Syndication) is a type of web feed[2] which allows users to access updates to online content in a standardized, computer-readable format. These feeds can, for example, allow a user to keep track of many different websites in a single news aggregator. The news aggregator will automatically check the RSS feed for new content, allowing the content to be automatically passed from website to website or from website to user.

4.6 MD5 library

MD5 is prone to length extension attacks. The MD5 algorithm is a widely used hash function producing a 128-bit hash value. Although MD5 was initially designed to be used as a cryptographic hash function, it has been found to suffer from extensive vulnerabilities. It can still be used as a checksum to verify data integrity, but only against unintentional corruption.
